DhanaRakshak is an AI-powered personal finance tracker built using Flask, MySQL, and Machine Learning concepts.

## Features
- User authentication (JWT)
- CSV / Excel transaction upload
- Expense & income tracking
- Interactive dashboard (charts & KPIs)
- AI-powered financial suggestions
- Anomaly detection & overspending alerts

## Tech Stack
- Backend: Flask (Python)
- Database: MySQL
- Frontend: HTML, Bootstrap, Chart.js
- Data Analysis: Pandas
- AI Logic: Rule-based + statistical analysis
